# /usr/local下目录的区别

| 目录                   | 内容      | 作用                                 |
| -------------------- | ------- | ---------------------------------- |
| `/usr/local/bin`     | 可执行程序   | 用户安装的软件的可执行文件（命令行程序）               |
| `/usr/local/lib`     | 库文件     | 编译好的 **静态库（.a）或动态库（.so）**，供链接器使用   |
| `/usr/local/include` | 头文件     | **C/C++ 的头文件**（.h/.hpp），供编译器在编译时引用 |
| `/usr/local/share`   | 架构无关的数据 | 文档、配置文件、pkgconfig、样例数据等，不依赖 CPU 架构 |
